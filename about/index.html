Silvester Hsu

我是一名致力于高效算法和GPU编程的开发者，专注于计算机视觉、深度学习和高性能计算的技术研究与应用。在并行计算、图像处理、推理引擎优化等方面有着丰富的经验，擅长利用CUDA进行性能优化，并对GPU内存管理、深度学习框架加速有深入理解。

在我的GitHub代码库中，您可以找到我在多个领域的开源项目，涵盖了从GPU内存管理工具、图像去畸变算法、到高效的模型推理引擎优化等方面的实现。曾为多个深度学习项目贡献代码，并在这些项目中成功提升了推理速度和计算效率。

技术栈

  • 编程语言: C++, Python, Bash
  • GPU编程: CUDA, TensorRT, cuDNN
  • 深度学习框架: TensorFlow, PyTorch, ONNX
  • 图像处理: OpenCV, Computer Vision (CV)
  • 数据库: SQLite
  • 操作系统: Linux (Ubuntu), Windows
  • 工具与环境: Git, Docker, Jenkins, CMake
  • 性能优化: SIMD, 多线程, 高性能计算（HPC）

我对算法优化、计算图设计和GPU加速有浓厚兴趣，尤其是在自动驾驶、图像识别等领域的应用。希望能与更多志同道合的开发者一起交流、合作，共同推动前沿技术的发展。
